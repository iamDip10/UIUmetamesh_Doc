How to run UIU-METAMESH

To run our software you have to install our software in your system. To do that follow the steps accordingly:

  1. Install the latest Python.
  2. Install MySQL database.
  3. Open a new terminal and type pip install -r requ.txt to install all dependencis.
  4. Then type python manage.py makemigrations and python manage.py migrate.
  5. Then type python manage.py runserver to open the server and ready to go.

How to create a account

For registering in our system,

  1. Click the signup button to create an account and fill the following fields. Enter your UIU Email Address to be an Authorized user.
  2. Once you have completed the signup process you will be redirected to the Landing page.
  3. Now enter your Email and Password to Log-In.

After login, you will be on the Homepage of our system. Here we have our features like:

  1. Profile Information: Users can see their information from the top left of the home page.
  2. Chat: The user can initiate chat by clicking on the chat icon.
  3. Notification: on the left bottom user can see the reaction notification.
  4. Notice: The user can see the real-time notice after clicking on the notice icon.
  5. Jobs: Available jobs news will appear here.
  6. Active User: on the right side, the user can see the active user.
  7. Profile: The user can go to the profile settings by clicking on the profile icon (user image) in the top right corner.
  8. Logout: The user can logout by clicking on the logout button.

How to Post

To publish a post,

  1. On the home page users can post by clicking the post (share your thoughts) option. Users can also categorize their posts accordingly by clicking the filter by category button and the dropdown will appear to choose from.
  2. Users can react by clicking on the upvote button and share their thoughts by clicking on the Enter a comment field.

How to Initiate Chat

  1. Click on the chat icon from the top middle section it will redirect to the real-time chat page.
  2. On the left side, there is an active user list.
  3. Click on the user to whom you want to start a conversation.
  4. After clicking on the user icon, the chat window will open on the right side.
  5. Now write your message in the enter message section and click the send button to send the message.
  6. Now you can see your message and reply from the other end user above.

How to see Notice

  1. Click on the notice icon, it will redirect you to the notice page where you can see the real-time notices from UIU.
  2. Click on the notice to learn more about the specific notice. It will redirect to the UIU website notice panel.

How to open/join a club

  1. Click on the club icon, it will redirect you to the club page.
  2. To create a new club click on the create club button and provide the information. Note: If you want to the admin check the admin box. If you want to make another user admin click on the email box and provide that user email.
  3. Now click on the create club button now your club is created. And you are redirected to the club page.
  4. Now click the Enter button to enter the club.
  5. Here you can post and create events.
  6. To post here follow the how-to post steps.
  7. To create an event click on the create new event button and provide the information and click on the create event button your event is created.
  8. Click on the view details button to view the details of the event.
  9. If you are the admin you can allow or reject members on the right side of the club page of the individual clubs.
  10. Click on the member under the Pending request section now you can approve or reject the user request to join the club.

How to Search Job

  1. To see the latest job news click on the Jobs. Now you will be redirected to the jobs page.
  2. Here you can see the latest jobs.
  3. Click on the job to learn more about a specific job.
